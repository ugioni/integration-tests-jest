# Getting Started

### Pactum docs:
 - [PactumJS](https://pactumjs.github.io/api/api/table-of-contents.html)

### Prerequisites:
 - NodeJS `v12.22.1`

### How to run?

Inside of the project folder run:

 1. `npm install --save-dev`
 1. `npm run ci`

After that you should see a `./output` folder with some `HTML` reports.
